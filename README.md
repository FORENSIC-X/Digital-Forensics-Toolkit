# DIGITAL-FORENSICS
My journey into Digital Forensics. This repository documents my learning and contains custom-built investigative tools, including secure file hashers, file signature analyzers and metadata extractors for analyzing digital evidence and ensuring data integrity. 

Building these tools on my own from scratch is an attempt on my part to understand what exactly do the professional tools used for these purposes actually do under the hood. 

### For each industry tool studied, a custom implementation was first built from scratch to understand the underlying mechanism before using the production tool.
